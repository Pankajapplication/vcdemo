# VideoCrypt Demo

A demo website showcasing VideoCrypt, a comprehensive video analytics and management platform. This demo illustrates key features for video content management, analytics, and recommendation systems.

## Features

- **Analytics Overview**: Dashboard with sample analytics data including views, plays, downloads, and charts.
- **Media Player**: HTML5 video player with sample content.
- **VOD Library**: Video-on-demand library with genre-based organization and sample videos.
- **Recommendation Engine**: Personalized video recommendations based on user behavior.
- **Media Asset Management (MAM)**: Centralized system for ingesting, managing, and securing media assets.
- **Responsive Design**: Dark-themed UI optimized for desktop and mobile viewing.

## Pages

- `index.html` - Analytics Overview Dashboard
- `media-player.html` - Media Player Interface
- `vod.html` - VOD Library Management
- `recommendation.html` - Recommendation Engine
- `mam.html` - Media Asset Management
- `loader.html` - Loading Animation Demo
- `home.html` - Alternative Home Page

## How to Run

1. Clone or download the project files.
2. Open `index.html` in your web browser.
3. Navigate through the different sections using the sidebar menu.

## Technologies Used

- HTML5
- CSS3 (Bootstrap, Custom Styles)

- Font Awesome Icons
- Google Charts
